# Zahni Mahir (زہنی ماہر)

An interactive, bilingual (Urdu & English) educational quiz application designed for children.

## Features
- **13 Educational Modules:** Including Science, Math, Islamic Studies, Logic, and more.
- **30 Levels per Module:** Total of 390 unique questions.
- **Child Safe:** No ads, no data collection, and runs entirely offline/locally.
- **Progress Tracking:** Automatically saves unlocked levels on the device.

## How to Play
1. Open the [Live Link](https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/) in any browser.
2. Select a category.
3. Solve levels to unlock the next ones.
4. Use "Add to Home Screen" on mobile to use it like an app.

## Safety & Security
This app uses static HTML/JavaScript. It does not communicate with any server, making it 100% safe for children.
